Say Yes...or No to the Dress

I want to do a project that see the effects of gender on dress code violations in schools throughout the United States. Using data gathered by The Pudding it gives me the opportunity to analyze the targets of dress code violations. I'm going to use a group cluster layout to see how violations are tied to gender.

Schools, schools, schools!!!

The data showcases 481 schools and their dress code policies. Through this page we are showcasing ways that schools punish students over their dress code, especially young women. We are showcasing ways that schools use rules and policies to penalize women. Let's use data to break down our data on dress code.
